请求配置

参数保持和后端接口一致，普通测试一般只需要选择文件和模式。

调用 POST /asr/file-sse，上传完整音频文件，直接消费 SSE 返回。
请求说明 
FormData: file, mode, audio_fs, chunk_size, chunk_interval, encoder_chunk_look_back, decoder_chunk_look_back, hotwords
调用 POST /asr/base64-sse，页面会把文件转成 Base64 后用 JSON 提交。
请求说明 
JSON: audio_base64, filename, mode, audio_fs, chunk_size, chunk_interval, encoder_chunk_look_back, decoder_chunk_look_back, hotwords
模拟实时流：创建会话，打开 SSE，再把音频转为 16k PCM16 后按片段 Base64 推送。

浏览器需要支持 WebAudio 解码当前音频格式。若某个格式失败，换 WAV 或 MP3 测试。

调用链路 
POST /asr/session -> GET /asr/sse/{session_id} -> POST /asr/chunk-b64/{session_id} -> POST /asr/end/{session_id}
